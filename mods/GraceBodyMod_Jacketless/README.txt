Grace Body Mod – Jacketless Outfit
==================================
Author: Keff0
Version: 1.0.0

This mod is designed for the Jacketless outfit. It removes the jacket
and applies the body rework.

Installation:
1. Install REFramework.
2. Copy .pak files to ~mods folder.
3. Select Jacketless outfit in-game.

Note: May conflict with other body mods; use only one at a time.
