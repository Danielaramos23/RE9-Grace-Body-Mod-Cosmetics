Grace Body Mod – Standard
==========================
Author: Keff0
Version: 1.0.0

This mod provides a natural, lore-friendly rework of Grace Ashcroft's body
proportions, inspired by character models from Ashley (RE4) and Eva.

Installation:
1. Make sure REFramework is installed (dinput8.dll in game folder).
2. Copy the .pak files from the "files" subfolder to:
   [RE9 installation folder]\RE9\Content\Paks\~mods\
3. If using Fluffy Mod Manager, import the mod through the manager.

Note: This mod does not affect gameplay. It is purely cosmetic.

For updates and more variants, visit Nexus Mods.
