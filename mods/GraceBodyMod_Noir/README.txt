Grace Body Mod – Noir Outfit
============================
Author: Keff0
Version: 1.0.0

This variant works with the Noir costume. It includes adjusted textures
to match the outfit's style.

Installation:
1. Ensure REFramework is installed.
2. Copy .pak files to ~mods folder as described in the main readme.
3. Activate the Noir outfit in-game to see the effect.

Compatible with the Standard version – choose one.
