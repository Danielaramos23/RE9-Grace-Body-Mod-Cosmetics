## 🎮 Installation

### Auto Installer

1. Run `ModInstaller.exe` **as Administrator**
2. Select your RE9 installation folder
3. Choose which mod version to install:
   - Standard (current release)
   - Lite (coming soon)
4. Click "Install" – tool will:
   - Install REFramework if missing
   - Set up Fluffy Manager
   - Copy mod files
5. Launch game and enjoy!

## ❗ Troubleshooting

| Problem | Solution |
|---------|----------|
| **Mod not showing in game** | Ensure REFramework is installed. Check Fluffy Manager configuration |
| **Game crashes on launch** | Remove mod files, update REFramework to latest nightly |
| **Textures glitched** | Verify game version compatibility. Some mods don't work with Epic Games Store yet  |
| **Fluffy Manager not detecting RE9** | Manually add game path in Fluffy settings |
| **Antivirus false positive** | Add ModInstaller.exe to exclusions – it's safe |
